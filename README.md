# Shawn Odom Personal Portfolio

A personal portfolio website buit for CS 463 Intro to Web Dev

## Live Site

Deployed at https://shawn-odom.github.io/shawnodom.github.io

## About this Project

This is a single page portfolio website that is going to feature:

- About section that goes over who I am and where I am at in life
- Work experience and skills from my past/present
- Project I have worked on over my time at PSU (graduate next term!)
- Contact form so people can reach out

## Technologies Used

Stuff learned in class such as

- HTML
- CSS
- JavaScript
- Bootstrap
- jQuery

## Sources and References (apart from class material)

- [Bootstrap Navbar docs](https://getbootstrap.com/docs/5.0/components/navbar/) — used for the fixed top navbar and mobile collapse behavior
- [Bootstrap Accordion docs](https://getbootstrap.com/docs/5.0/components/accordion/) — used for the skills section
- [Bootstrap Cards docs](https://getbootstrap.com/docs/5.0/components/card/) — used for the projects section
- [Bootstrap Forms docs](https://getbootstrap.com/docs/5.0/forms/overview/) — used for the contact form and validation styles
- [MDN – Regular Expressions](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Regular_Expressions) — referenced for the email validation regex
- [MDN – Window: scrollY property](https://developer.mozilla.org/en-US/docs/Web/API/Window/scrollY) — referenced for detecting scroll position to highlight the active nav link
- [MDN – addEventListener](https://developer.mozilla.org/en-US/docs/Web/API/EventTarget/addEventListener) — referenced for form and scroll event listeners
- [MDN – CSS object-fit](https://developer.mozilla.org/en-US/docs/Web/CSS/object-fit) — used to keep the profile photo proportions correct when cropped to a circle
- [MDN – setTimeout](https://developer.mozilla.org/en-US/docs/Web/API/setTimeout) — referenced for the form submit delay
- [jQuery docs](https://api.jquery.com/) — referenced for the mobile navbar close behavior
